# Asymmetric-ARDL-analysis-
this repository contains R-codes for asymmetric ARDL analysis
1 R-codes for data generating process, (white noise, AR=0.5, AR=0.8, AR=0.95 and Random Walk)
2 R-codes for the ARDL analysis with ardl function
3 R-codes for the ARDL analysis with manual calculation
4 R-codes for the Monte Carlo simulation for the ARDL analysis with I(0): y is randam walk, but x is white noise, I(1) y and x are white noise 
5 R-codes for the Asymmetric ARDL analysis with ardl function
6 R-codes for the Asymmetric ARDL analysis with manual calculation
7 R-codes for the Monte Carlo simulation for the Asymmetric ARDL analysis with I(0): y is randam walk, but x is white noise, I(1) y and x are white noise
